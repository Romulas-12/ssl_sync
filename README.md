# SSL Sync - Home Assistant Add-on Repository

## Встановлення

Додайте цей репозиторій до Home Assistant:

1. Settings → Add-ons → Add-on Store
2. ⋮ (три крапки) → Repositories
3. Додайте URL репозиторію
4. Знайдіть "SSL Sync" та встановіть
